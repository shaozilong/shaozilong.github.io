To be able to do debuging in release mode with Visual Studio, do the following settings : 1. C/C++ -> Général -> Format des informations de débogage >> Base de données du programme 2. Editeur de liens -> Débogage -> Générer des infos de débogage >> Générer des informations de débogage (/DEBUG) Optionally (note necessary to disable the optimization) 3. C/C++ -> Optimisation -> Optimisation >> Désactivé
