1. Creat a new empty folder (repo) as repository to store and track the code 2. Go into the empty folder (repo), right-click->TortoiseSVN->Create repository here, copy the URL 3. Import the initial code to the repository, by right-click on the code file, select TortoiseSVN->Import, paste the URL of the repository 4. Create a new folder to check out the code from the repository, right-click->SVN CheckOut, make sure of the right URL
